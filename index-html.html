O que é HTML?

HTML (Hyper Text Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ele consiste em uma série de elementos que são usados para estruturar e formatar o conteúdo da página.

Aqui estão alguns conceitos fundamentais do HTML:

  • Elementos: Os elementos HTML são as peças fundamentais da linguagem e são representados por tags. Uma tag HTML é cercada por colchetes angulares (<>) e contém o nome do elemento. Por exemplo, a tag '<p>' é usada para criar um parágrafo.
  • Tags de abertura e fechamento: A maioria dos elementos HTML requer uma tag de abertura e uma tag de fechamento. A tag de abertura indica o início do elemento, enquanto a tag de fechamento indica o seu fim. Por exemplo: 
                                <p>Este é um parágrafo.<p>
  • Atributos: Os atributos fornecem informações adicionais sobre os elementos HTML. Eles são especificados na tag de abertura e têm um nome e um valor. Por exemplo, o atributo src é usado para especificar a localização de uma imagem em uma tag' <img>'. Exemplo: 
    •                         <img src="caminho/para/imagem.jpg" alt="Descrição da imagem" >

Estrutura básica de uma página HTML: Toda página HTML deve ter uma estrutura básica. Aqui está um exemplo:

            
                
                    <!DOCTYPE html>
                    <html>
                        <head>
                            <title>Título da Página</title>
                        </head>
                    <body>
                        <h1>Meu Primeiro Cabeçalho</h1>
                        <p>Meu Primeiro Parágrafo.</p>
                        </body>
                    </html>

A declaração <!DOCTYPE html> define que este documento é um documento HTML5.

O elemento <html> é o elemento raiz de uma página HTML.

O elemento </head> contém informações meta sobre a página HTML.

O elemento <title> especifica um título para a página HTML (que é exibido na barra de título do navegador ou na guia da página).

O elemento <body> define o corpo do documento e é um container para todo o conteúdo visível, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

O elemento <h1> define um cabeçalho grande.

O elemento <p> define um parágrafo.

  • Outros elementos HTML: Além dos elementos mencionados acima, o HTML possui uma ampla variedade de elementos para diferentes finalidades. Alguns exemplos incluem: <a> para links, <img> para imagens, <ul> e <li> para listas, <table> para tabelas , <form> para formulários, entre outros.

Esses são apenas conceitos básicos do HTML. À medida que você avança, pode aprender mais sobre formatação de texto, estilos CSS, elementos avançados, acessibilidade e outras práticas recomendadas para criar páginas da web de alta qualidade.

Lembre-se de que o HTML é apenas a estrutura da página, e o estilo e o comportamento podem ser adicionados usando CSS e JavaScript, respectivamente.

Aninhamento de elementos: É possível aninhar elementos HTML, o que significa colocar um elemento dentro de outro.

Exemplo:

                    
                        <div>
                            <h2>Título do Div</h2>                    
                            <p>Este é um parágrafo dentro do div.</p>
                        </div>

Nesse exemplo, o elemento '<h2>' e o elemento '<p>' estão aninhados dentro do elemento '<div>'.

Comentários: Você pode adicionar comentários ao seu código HTML para fins de documentação ou para explicar partes específicas do código. Os comentários não são exibidos na página, mas são úteis para outros desenvolvedores ou para você mesmo ao revisar o código. Os comentários em HTML são escritos entre <!-- e -->.

Exemplo:

                    <!-- Este é um comentário em HTML -->

Elementos vazios: Alguns elementos HTML não possuem conteúdo e não precisam de uma tag de fechamento. Eles são chamados de elementos vazios ou autossuficientes. Por exemplo, a tag '<br>' é usada para inserir uma quebra de linha e não requer uma tag de fechamento.

                    
                        <br>

Links: O elemento '<a>' é usado para criar links em uma página HTML. Ele requer o atributo "href", que especifica o destino do link.

Exemplo:

                    <a href="https://www.example.com">Clique aqui para visitar o site</a>

Ao clicar no link, o usuário será redirecionado para o URL especificado no atributo "href".

Listas: Você pode criar listas ordenadas '<ol>' ou listas não ordenadas '<ul>' em HTML. Cada item da lista é representado pela tag '<li>'.

Exemplo:

                    
                        <ul>
                            <li>item 1</li>
                            <li>item 2</li>
                            <li>item 3</li>
                        </ul>

Isso criará uma lista não ordenada com três itens.
